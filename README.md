## Installation

### Create a .venv

```bash
python3 -m virtualenv -p Python39 .venv
```

### Activate the .venv

```bash
#For Windows
source .venv/Scripts/activate
#For Mac
source .venv/bin/activate
```

## Using

Run the `presidio.ipynb` notebook.
